    Spoon Server Requirements

    Spoon Server currently supports the following server platforms:

    • Microsoft Windows® 7 Server
    • Microsoft Windows Server® 2008
    • Microsoft Windows® Vista
    • Microsoft Windows Server® 2003

    On the client side, Spoon supports all of the above server platforms, plus:

    • Microsoft Windows® 7
    • Microsoft Windows® XP
    • Microsoft Windows® Vista

    Spoon supports applications built for both the x86 and x64 processor architectures, and both 32- and 64-bit versions of the supported operating systems.

    Spoon Server requires adequate bandwidth to service the expected number of concurrent user streams (approximately the size of the virtual application package multiplied by the number of expected concurrent streams).  No significant bandwidth is utilized once the virtual application package has been completely transferred. 

    Spoon recommends an effective transfer rate of at least 1.5Mbps.
